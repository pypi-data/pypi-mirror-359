This module allows you to associate ZIPs to code programs and code
programs territories. It also adds those fields to the partner address
autocompletion

Usage
=====

To use this module, you need to:

1. Go to **Contacts > Configuration > Zips** and fill the code programs
   and code program territories of the Zips.
2. Go to a res partner form and fill the location completion, the code
   program and code program territory will also be autocompleted.
