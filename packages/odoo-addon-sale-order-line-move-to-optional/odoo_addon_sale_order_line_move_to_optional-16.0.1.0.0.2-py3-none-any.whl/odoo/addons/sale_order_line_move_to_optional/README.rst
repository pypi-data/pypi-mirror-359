This module enables moving sale order lines to the optional products
section, while removing them from the main order lines.

**Table of contents**

.. contents::
   :local:

Usage
=====

You can move order lines to the optional products section only when the
sales order is in Draft or Quotation Sent state

- Open the sales order
- Click on "Move to optional" next to the order line you want to move
- The product will be removed from the order lines and added to the
  Optional Products section
